### 主要文件目录结构

```
frontend            主路径
└─ src              “@”指向的目录
   ├─ App.vue       启动页       
   ├─ main.js       主函数
   ├─ views         单页面目录
   │  ├─ FavoritesPage.vue
   │  ├─ LoginPage.vue
   │  ├─ RegisterPage.vue
   │  ├─ SessionPage.vue
   │  ├─ SettingPage.vue
   │  └─ UserInfoPage.vue
   ├─ router        页面路由配置目录
   │  └─ index.js
   ├─ layout        主页面布局
   │  └─ AppLayout.vue  实现顶栏+侧栏
   └─ components    组件目录
      └─ Navigation.vue 实现侧栏


```
